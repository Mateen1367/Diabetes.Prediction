# Diabetes Prediction Project

## Overview
This project uses machine learning to predict whether a patient has diabetes based on health-related features.

It is a binary classification problem where:
- 0 = No Diabetes
- 1 = Diabetes

---

## Dataset
The dataset includes medical features such as:
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

Target Variable:
- Outcome

---

## Objective
The goal of this project is to:
- Perform data analysis and visualization
- Handle missing values and clean the data
- Build a machine learning model
- Evaluate the model performance

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Project Steps
1. Data Loading and Exploration
2. Data Cleaning (handling missing values)
3. Data Visualization
4. Feature Scaling
5. Model Training (Logistic Regression)
6. Model Evaluation

---

## Results
The model was able to predict diabetes with reasonable accuracy.  
Key features such as glucose and BMI were strong indicators of diabetes.
